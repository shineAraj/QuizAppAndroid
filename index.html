Android Quiz App

A Quizzing application

Download this project as a .zip file Download this project as a tar.gz file

A Quizzing application using which a User can attempt an objective Quiz from a list of Quizzes. Each quiz has 5 questions, each with same or different weightage. The user selects one option (out of four options), and only then can s/he move forward to the next question. At the end of each quiz, the student is shown the answer.
